## Local Debugger Setup

To set up a local debugger in **VS Code**, follow these steps:

1. Create a folder named **.vscode** in your project directory.
2. Inside the **.vscode** folder, create a file named **launch.json**.
3. Copy and paste the following content into `launch.json`:

```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "node",
            "request": "launch",
            "name": "Launch Program",
            "skipFiles": [
                "<node_internals>/**"
            ],
            "program": "${workspaceFolder}/server.js"
        }
    ]
}
